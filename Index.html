Projects

Project One: The Check Project

This project was a roguelike version of chess, including deckbuilder elements. You would go from battle to battle, each time you win you continue forward, but if you fail then you get reset to the beginning. I was the main code developer for this project, and I managed to do all of the logic and basically setting everything up in the Godot software by myself. I did however have the support of friends, who had made the great piece models, and the 'tree' model.
